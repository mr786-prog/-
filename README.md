## 🚀 How to Use

### Option 1: Local Installation
1. Download the entire `arwaesthetics-website` folder
2. Open `index.html` in your web browser
3. No additional dependencies required - it's pure HTML/CSS/JavaScript

### Option 2: Web Server
1. Upload the entire folder to your web hosting
2. Navigate to the folder URL in your browser
3. The website will load and work perfectly

### Option 3: GitHub Pages
1. Push the folder to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Access via `yourusername.github.io/arwaesthetics-website`

## 🔧 Customization Guide

### Change Colors
Edit the CSS variables in the `<style>` section:
```css
:root {
    --primary: #00d9ff;      /* Main cyan color */
    --secondary: #ff006e;    /* Magenta accent */
    --tertiary: #8338ec;     /* Purple */
    --accent: #ffbe0b;       /* Gold */
}
```

### Change Contact Information
Find the contact section and update:
- Phone number in href and display
- Email address in href and display
- Location details

### Add More Portfolio Items
Add new `<div class="portfolio-item">` blocks in the portfolio section with:
```html
<div class="portfolio-item">
    <img src="assets/images/your-image.jpg" alt="Description">
    <div class="portfolio-overlay">
        <div class="portfolio-info">
            <h3>Project Title</h3>
            <p>Category</p>
        </div>
    </div>
</div>
```

### Modify Animations
All animations are defined in the CSS `@keyframes` sections. Adjust duration, delay, and transform values as needed.
